# JanSetu: Premium Municipal Command Center - Frontend

Welcome to the frontend of **JanSetu**, a state-of-the-art municipal management platform designed with a **"Royal Indian"** digital aesthetic. 🇮🇳✨

## ✨ The "Royal Indian" Experience

JanSetu isn't just a CRM; it's a culturally-inspired, high-performance interface built for the modern Indian citizen and government official.

### 🖼️ Design Highlights
*   **Premium Glassmorphism**: Utilizes `backdrop-blur-2xl` architecture for a sophisticated, "floating" UI that feels modern and lightweight.
*   **Rangoli Identity**: Subtly integrated SVG Rangoli motifs (`RANGOLI_PATTERN`) that connect the digital platform with India's rich cultural heritage.
*   **JanSetu Palette**: A curated color system using **Saffron (#FF9933)**, **Gold (#F4B400)**, and **Deep Indigo (#2B6CEE)**.
*   **Mobile-First Glass Cards**: The "Community Pulse" feed uses an interactive grid of cards, optimized for one-handed operation on mobile devices.
*   **Micro-Animations**: Staggered entry animations (via `framer-motion`) and celebratory confetti feedback (via `canvas-confetti`) provide a premium user experience.

## 🛠️ Tech Stack
*   **Core Framework:** Next.js 14+ (App Router)
*   **Styling:** Tailwind CSS (Vanilla)
*   **UI Components:** Shadcn UI (Atomic Design)
*   **Animations:** Framer Motion & Canvas-Confetti
*   **Data Visualization:** Recharts
*   **Mapping:** Ola Maps API & MapLibre GL JS
*   **State Management:** React Context API

## 🚀 Key Features
1.  **Community Pulse**: View and support local civic issues through an interactive card-based feed.
2.  **City Insights (Analytics)**: High-resolution data bars and pulsing metric highlights for administrative transparency.
3.  **Precision Reporting**: 100% jurisdictional accuracy using Ola Maps reverse-geocoding and PIN-code "snapping" logic.
4.  **Evidence Dashboard**: High-resolution image previews and status audit trails for every civic issue.

## 📦 Getting Started
1.  Clone the repository.
2.  Run `npm install`.
3.  Configure your `.env.local` with:
    *   `NEXT_PUBLIC_API_BASE_URL`
    *   `NEXT_PUBLIC_OLA_MAPS_API_KEY`
    *   `NEXT_PUBLIC_IMGBB_API_KEY`
4.  Launch the dev server with `npm run dev`.
